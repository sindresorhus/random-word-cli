# random-word-cli

> Get a random English word


## Install

```
$ npm install --global random-word-cli
```


## Usage

```
$ random-word --help

  Example
    $ random-word
    ferriferous
```


## Tip

Use it to generate project/release names:

```
$ echo $(random-word)-$(random-word)
blacksnake-nautics
```
