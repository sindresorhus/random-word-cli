# random-word-cli

> Get a random English word

## Install

```sh
npm install --global random-word-cli
```

## Usage

```
$ random-word --help

  Example
    $ random-word
    ferriferous
```

## Tip

Use it to generate project or release names:

```sh
echo $(random-word)-$(random-word)
blacksnake-nautics
```
